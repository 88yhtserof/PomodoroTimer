# PomodoroTimer
뽀모도로 타이머 서비스
DatePicker를 사용해 시간을 설정하고 Progress와 타이머로 시간을 확인할 수 있다.   
일시정지, 시작, 취소 기능을 사용할 수 있다.   
애니메이션 기능을 사용해 타이머가 실행되는 동안 이미지가 움직인다.
